# WireGuard SOC Lab

SOC monitoring project based on:

- WireGuard VPN
- auditd
- Python parser
- Telegram alerts

## Features

- Detect VPN config changes
- Detect private key access
- Detect WireGuard command execution
- Real-time Telegram notifications

## Technologies

- WireGuard
- auditd
- Python
- systemd
- Telegram Bot API
